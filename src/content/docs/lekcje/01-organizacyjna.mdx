## Skąd się wziął Linux

**Richard Stallman** — programista, założyciel [Free Software Foundation](https://www.fsf.org/) i autor
projektu **GNU** (1983). Idea: stworzyć kompletny uniksopodobny system operacyjny zbudowany wyłącznie
z wolnego oprogramowania. Do końca lat 80. większość komponentów GNU była gotowa — brakowało jądra.

**Linus Torvalds** wypuścił pierwsze publiczne wydanie jądra **Linux** **25 sierpnia 1991** roku (mailing
list [comp.os.minix](https://en.wikipedia.org/wiki/Comp.os.minix)). W połączeniu z narzędziami GNU
powstała kompletna platforma, którą dziś nazywamy GNU/Linuxem (lub potocznie po prostu Linuxem).

Cała baza kodu — od jądra po userland — jest na licencjach typu GPL/MIT/BSD. Oznacza to wolność użycia,
modyfikacji i redystrybucji bez opłat licencyjnych.

## Trzy pierwsze komendy

Komendy nawigacji po systemie plików — fundament pracy z terminalem.

```bash title="Terminal"
ls          # wylistuj zawartość bieżącego katalogu
ls -la      # długi format + ukryte pliki (zaczynające się od kropki)

pwd         # print working directory — wypisz ścieżkę bieżącego katalogu

cd ..       # przejdź jeden katalog wyżej
cd ~        # przejdź do katalogu domowego (alias: cd bez argumentu)
cd -        # wróć do poprzedniego katalogu
```

:::note[Uzupełnienie 2026]
W 2026 standardem dla nowych instalacji jest **Btrfs** lub **ZFS** z snapshots zamiast tradycyjnego ext4.
Ubuntu domyślnie nadal proponuje ext4 dla `/`, ale instalator oferuje ZFS jako opcję eksperymentalną.
:::
